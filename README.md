# Lachen Platz

Lachen platz is a small private forest monitoring project in Nagano, Japan.

This project aims to document and analyze ecological and seasonal changes using:

- camera traps
- fixed-point photography
- low-cost microclimate sensors
- field notes
- GIS
- Python-based visualization

## Current focus

The first phase focuses on:

- mammal activity recorded by trail cameras
- basic microclimate monitoring
- fixed-point photographic records
- site mapping with masked spatial information
- reproducible analysis workflows

## Data policy

Exact coordinates, land parcel information, raw videos, EXIF-containing original images, and location-sensitive files are not publicly released.

Only processed, anonymized, spatially masked, or sample datasets will be shared in this repository.

## Status

This project is in an early setup phase.
