# Data Policy

This repository does not publish location-sensitive or privacy-sensitive data.

## Not publicly released

The following data must not be committed to this public repository:

- exact coordinates
- land parcel information
- private boundary polygons
- raw trail camera videos
- EXIF-containing original photos
- images or videos containing people, vehicles, license plates, or neighboring properties
- records of sensitive species with precise locations

## Publicly shareable data

The following data may be released after review:

- processed camera trap event tables
- anonymized or spatially masked site metadata
- aggregated sensor summaries
- low-resolution images with EXIF metadata removed
- generated figures
- analysis notebooks
- reproducible Python scripts
